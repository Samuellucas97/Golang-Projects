## Essential Commands


|          Action          |           Command           |
|:-------------------------:|:-------------------------------:|
|         Executing         |         `go run main.go`        |
|  Generate build on Linux  |        `go build main.go`       |
| Generate build on Windows | `GOOS=windows go build main.go` |
